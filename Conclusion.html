Summary

In summary, we use causal inference about changes in housing prices and birth rates.

We simplified the causal model of housing price changes and birth rate changes, and added other variables to draw the DAG. We identified the results using backdoor criteria and backdoor adjustments. We then used linear outcome regression with interaction terms to estimate the ATE, and finally considered a sensitivity analysis for unmeasured factors using Manski bounds.
